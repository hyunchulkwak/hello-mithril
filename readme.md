# Hello Mithril
Using MithrilJS with JSX syntax

## Getting Started
Use npm or [yarn](https://yarnpkg.com/) to install depedencies and run `npm start` or `yarn start`.
```
$ yarn install
$ yarn start
```
```
$ npm install
$ npm start
```
